# shrikuberantiffenhome
Responsive landing page for Shri Kuberan Tiffen Home, showcasing South Indian vegetarian meals with menu cards, contact info, location map, and online ordering links. Built with HTML, CSS, Bootstrap 5.

Menu cards for Idly, Ven Pongal, Tomato Rice, Puliyotharai, Malli Rice, Chapati

Contact information with FSSAI number

Google Maps location embed

Navbar with logo branding

Order button linked to Zomato

Built with HTML5, CSS3, and Bootstrap 5 for responsiveness

Optional AOS animations for smooth scrolling effects
